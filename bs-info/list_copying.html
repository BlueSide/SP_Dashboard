1 How to copy a list from one site to another

1.1 Activate Scripting via the SharePoint Online Management Shell

Scripting is now activated for around 24 hours. Everybody says it's bad practice, but since nobody gives another solution or alternative to exporting List Templates, this seems the least risky way.

1.2 Now we can go to List Settings in the source list and click 'Save list as template'

1.3 The Template is now available through Site Settings->List Templates

1.4 Click on the template to download the STP file

1.5 Open the STP file with any Archive tool (7-zip, WinRAR)

1.6 Extract manifest.xml

1.7 If needed, change the language codes to the destination site's language in manifest.xml

1.8 Re-archive to an STP file

1.8.1 Add the following contents:

; DIAMOND Directive File (.ddf)
.OPTION EXPLICIT
; Generate errors on variable typos
.Set CabinetNameTemplate=mylist.stp
.Set Cabinet=on
.Set Compress=on

;The files specified below are stored, compressed, in the cabinet file
manifest.xml

1.8.2 In a shell execute the following command

makecab /f definition.ddf

1.8.3 The outputted file is in a folder named 'disk1' and is probably called 'mylist.stp'

1.9 On the destination site go to Site Settings->List Templates

1.10 Click 'Upload Document' and upload the new STP file

1.11 it should now appear in the list with the right language and also in 'Add an App'
